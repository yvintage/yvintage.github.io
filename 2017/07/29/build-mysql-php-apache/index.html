搭建mysql + php + Apache环境

发表于 2017-07-29 | 分类于 安装

本教程难度级别：1颗星
平台：win10
介绍：本人后端小白，基于我在安装的过程中百度谷歌的我头都大了，所以打算写一个小白教程来给后来的小白们铺个路。

安装mysql

对于小白来说，建议下载mysql的完整安装包，不需要internet链接。

打开mysql的官网,点击downloads -> Community -> mysql on windows -> MySQL Installer.


点击MySQL Installer，移动到页面最下方，下载完整版安装包。
注意：mysql安装程序是32位的，但是会同时安装32位和64位的程序。


下载完成后，双击mysql安装包（.msi）文件。
